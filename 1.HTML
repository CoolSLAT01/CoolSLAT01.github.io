Basic Information About HTML

What is HTML?

HTML stands for HyperText Markup Language. It is used to create web pages and structure the content.

Why we use HTML?

HTML is used to display text, images, links and other content on the internet.Also it helps us to make websites.

Basic Structure

Every HTML page has some main parts:

.html - main tag
.head - contains title
.body - contains visible content

Some Common Tags

.p - paragraph
.a - link
.img - image

Conclusion

HTML is the first step to learn web development.
